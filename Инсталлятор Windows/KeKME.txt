Приложение «Кекнайзер» предназначено для планирования своего рабочего времени. При помощи приложения пользователь
- добавляет свои задачи, у которых есть срок выполнения и сложность выполнения. 
- Задачи сохраняются. 
- Пользователю добавляет свое занятое время, чтобы приложение посчитало его свободное время.
- На основе полученных данных приложение распределяет время под задачи пользователя.
- Пользователь может смотреть распределенное время.

Подробнее в справке к приложению.
